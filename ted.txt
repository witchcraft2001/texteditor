Text Editor v1.1 (Apr.2026)
based on ZX/IBM Text Editor sources
ver.1.0 (Oct.1993) by Hohlov Oleg
ported for Sprinter by Mikhalchenkov Dmitry

Current limits:
Maximum text file length is 39Kb
Maximum line length is 255 bytes

Hot keys in edit mode:
  F2 - Save
  Shift+F2 - Save As
  F3 - Open (Load)
  F10 - Main menu
  ESC - Command menu
  Ctrl+C - Copy block to clipboard file
  Ctrl+X - Cut block to clipboard file
  Ctrl+V - Paste block from clipboard file
  Ctrl+PgUp - Jump to beginning of text
  Ctrl+PgDn - Jump to end of text
  Ctrl+Y - Delete current line

Functions Command menu:
  B - Mark block begin
  E - Mark block end
  C - Copy block
  D - Delete block
  M - Move block
  Q - Reset block
  S - Search
  R - Replace
  L - Delete line
  G - Graphics On/Off
  J - Jump to line ...
  0..9 - Put decimal code
